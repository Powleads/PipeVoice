Questions

Windows says "unrecognised app" — is it safe?

Yes. That SmartScreen warning appears for any app that isn't code-signed (a paid certificate). Click More info → Run anyway. WisprLite is open source — you can read every line on GitHub.

Do I need an API key?

For the cloud engines (OpenAI / Deepgram), yes — you bring your own, so you pay providers directly at cost (a few cents a day). Prefer zero keys? Use the offline local engine — it runs entirely on your machine for free.

Is my voice uploaded anywhere?

Only to the engine you choose. Cloud engines send audio to OpenAI/Deepgram for transcription. The local engine never sends anything off your computer. WisprLite itself has no servers and no telemetry.

Which engine should I pick?

OpenAI Whisper for best accuracy, Deepgram for live words as you speak, local for full privacy / offline. You can switch in the tray menu any time.
